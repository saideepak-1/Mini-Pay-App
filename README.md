# 💸 MyPay - Payment App (HTML/CSS/JS)

A lightweight payment web app inspired by PhonePe and Google Pay. Built with **HTML**, **CSS**, and **JavaScript**, this app simulates basic wallet functionalities like sending money, mobile recharge, balance tracking, and transaction history.

---

## 📱 Features

- 💰 **Wallet Balance**: Real-time wallet balance updates
- 🧾 **Transaction History**: See recent money transfers and recharges
- ✅ **Input Validations**:
  - Phone number must be 10 digits
  - Name and amount fields are mandatory
- 🔊 **Audio Feedback**: Transaction success is confirmed with a sound
- 🖥️ **Responsive Design**: Clean and mobile-friendly UI

---
